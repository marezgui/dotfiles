# Malek's dotfiles

## How to get started

```sh
sh dotfiles.sh mac
sh dotfiles.sh zsh
sh dotfiles.sh home
sh dotfiles.sh brew
sh dotfiles.sh cask
```

## What's included

- gitconfig
- vimrc
- zshrc
- brew dependencies
- macos preferences

## Notes
