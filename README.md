# Malek's dotfiles

## Prerequisites

- Install [homebrew](https://brew.sh)
```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Setup
```sh
sh dotfiles.sh mac
sh dotfiles.sh zsh
sh dotfiles.sh home
sh dotfiles.sh brew
```

## What's included

- gitconfig
- vimrc
- zshrc
- brew dependencies
- macos preferences

## Notes
